## A1

opcode 有 64 条，则需要 6bit 才能表示完整。
register 有 56 个，也需要 6bit 才能表示完整。
则 IMM 可⽤位数为： 32 - 6 - 6 - 6 = 14bit
则表示范围： $-2^{13}<IMM<2^{13}-1$

|   **X**   | Does the program halt? | Value stored in `R0` |
| :-------: | :--------------------: | :------------------: |
| 000000010 |          Yes           |          2           |
| 000000001 |          Yes           |          3           |
| 000000000 |          Yes           |          6           |
| 111111111 |           No           |          --          |
| 111111110 |          Yes           | x8004 ($-2^{15}+4$)  |
